# Протокол встречи — Обновление структуры и функционала торгового бота
Дата: 3 февраля 2026 г.

Участники: один спикер (имена не указаны)

Ключевые решения

1) Модель распространения
- Бот условно-бесплатный: 3 бесплатных торговых сигнала в день.
- Пробного периода не будет; после трёх бесплатных сигналов показываем предложение снять ограничения.
- Для снятия ограничений пользователь обязан зарегистрироваться на платформе Pocket Option по реферальной ссылке — после успешной регистрации лимиты снимаются.

2) Структура сигналов
- Кнопки Long и Short сохраняются.
- Long: генерируются ИИ-моделями через OpenRouter; сопровождаются подробной аналитикой.
- Short: быстрые статистические сигналы на основе данных из базы; бот собирает статистику и самообучается.
- Поддерживаются все активы Pocket Option, включая ОТС — требуется парсинг доступных активов.

3) Взаимодействие и анализ скриншотов
- Пользователи смогут отправлять скриншоты для анализа.
- Двухуровневая система моделей:
  - Тяжёлые модели: анализ скриншотов (оценка банка и т.п.), результаты сохраняются в базе.
  - Лёгкие модели: диалог с пользователем, подтягивают данные из базы и отвечают на общие вопросы.

4) Доп. функционал
- Банк-менеджмент: управление банком через бота; советы ИИ и рекомендации по размерам ставок на основе анализа скриншотов.
- Админ-панель: добавление админов, выдача VIP, бан/удаление пользователей — функционал сохраняется.
- Интерфейс бота остаётся прежним.
